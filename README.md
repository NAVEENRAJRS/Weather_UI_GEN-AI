# 🌤 Weather Forecast App

A Weather Forecast application built using Python and Streamlit.

## Features

- Live weather information
- Search weather by city
- Temperature
- Feels Like
- Humidity
- Wind Speed
- Weather Condition

## Technologies Used

- Python
- Streamlit
- Requests
- OpenWeatherMap API

## Installation

git clone https://github.com/yourusername/weather-forecast-ui.git

cd weather-forecast-ui

pip install -r requirements.txt

streamlit run app.py
